 Portfolio Risk Analysis (Power BI)

 Overview
This project analyzes Growth, Balanced, and Preservation portfolios
using Power BI across multiple risk and return metrics.

 Metrics Included
- CAGR
- Volatility
- Maximum Drawdown
- Worst Monthly Return
- Portfolio Value Over Time
- Market Regime Analysis

Tools
- Power BI
- Microsoft Excel

The Growth portfolio delivers the highest long-term returns but with
greater volatility and deeper drawdowns, while the Balanced portfolio
offers the best risk-adjusted performance.
